# giorgio-webseite - Premium Trinkflasche

Eine elegante, moderne Website für eine Premium-Trinkflasche mit minimalistischem Design in Weiß und Grau.

## 🎨 Features

- **Responsive Design** - Funktioniert perfekt auf allen Geräten (Desktop, Tablet, Mobile)
- **Modernes Layout** - Sauberes und minimalistisches Weiß-Grau-Design
- **Sektionen**:
  - Hero Section mit Produktbild-Platzhalter
  - Produkt-Details (4 Highlights)
  - Features Liste
  - Kontakt-Formular
  - Sticky Header mit Navigation

- **Interaktivität**:
  - Smooth Scrolling Navigation
  - Hover-Effekte
  - Scroll-Animationen
  - Funktionales Kontakt-Formular

## 📁 Dateistruktur

```
giorgio-webseite-/
├── index.html       # Hauptseite
├── style.css        # Styling (Weiß/Grau Theme)
├── script.js        # JavaScript für Interaktivität
└── README.md        # Diese Datei
```

## 🚀 Verwendung

1. **Produktbild hinzufügen**:
   - Laden Sie Ihr Trinkflaschen-Bild hoch
   - Benennen Sie es `placeholder-bottle.jpg` (oder passen Sie den Namen in `index.html` an)

2. **Inhalt anpassen**:
   - Öffnen Sie `index.html` und passen Sie Texte nach Bedarf an
   - Ändern Sie Farben in `style.css` (`:root` Variablen)

3. **Mit GitHub Pages hosten**:
   - Gehen Sie zu Repository Settings → Pages
   - Wählen Sie Branch: `main`
   - Ihre Website ist dann erreichbar unter: `https://giorgioking.github.io/giorgio-webseite-/`

## 🎨 Farben

Das Design verwendet folgende Farbpalette:
- **Weiß**: `#ffffff`
- **Hellgrau**: `#f5f5f5`
- **Mittelgrau**: `#d3d3d3`
- **Dunkelgrau**: `#808080`
- **Sehr Dunkelgrau**: `#333333`

## 📱 Responsive Breakpoints

- **Desktop**: 1200px+ (Grid Layout)
- **Tablet**: 768px - 1199px
- **Mobile**: < 768px (Single Column)

## 💡 Tipps

- Verwenden Sie hochwertige Produktfotos

- Testen Sie auf verschiedenen Geräten
